## Set up steps:
1. go to /face-robot-01/
2. Compile with colcon: ```colcon build```
3. Open another terminal in root mode: ```sudo -s```
4. Install: ```source install/setup.bash```
5. Run ros2 with launch file: ```ros2 launch mp_input mp.launch.py```
